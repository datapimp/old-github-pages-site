What is Reloaded?

Reloaded is a tool written in node.js which allows you to pipe code snippets in javascript or css directly into a running browser session without refreshing the page, or requiring any special browser plugins.

How does it work?

Reloaded typically gets run from the root of a big web project you are working on. Currently it only works nicely with Rails but I am in the process of adding support for all major web frameworks, and even allowing you to configure it however you like.

Reloaded watches the project for changes to javascript and css, and automatically notifies the browser when that happens which triggers a code reload.

This happens thanks to socket.io which provides websocket like functionality across a wide range of browsers.

You are also able to instruct the browser to run code through a command line utility, which is useful for piping snippets of code or injecting assets into the browser session.
